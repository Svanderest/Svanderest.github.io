Fire Training Portal

The Fire Training Portal is a scheduling and record keeping system for the training division of the St Catharines Fire Department. The portal is an ASP.NET web application using the MVC design pattern and Entity Framework. It was developed to replace an existing system using MS Access that was no longer fit for purpose.

Skill Used: C#, .NET Framework, Entity Framework, SQL Server, HTML/CSS, jQuery

Feekeeper

Feekeeper is a tool developped for the City of St Catharines to update municipal rates and fees, mostly related to building permits. It is an ASP.NET web application connected to the same Oracle database as the municipal permitting application, CSDC Amanda. The tool was developed to accomodate a version upgrade to CSDC Amanda which would otherwise have broken the exisinting fee updating process. Development of this application also included a much needed audit of existing fee procedures.

Skills Used: C#, .NET Framework, Oracle, Entity Framework, HTML/CSS, jQuery
